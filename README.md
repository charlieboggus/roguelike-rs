# roguelike-rs
A roguelike game written in Rust using [tcod-rs](https://github.com/tomassedovic/tcod-rs).

## Controls:

| Keybinding | Game Action |
|:----------:|:-----------:|
| `w` |Move player upwards |
| `s` | Move player downwards |
| `a` | Move player left |
| `d` | Move player right |

## Requirements:
- [Rust](https://www.rust-lang.org/en-US/install.html)
- libsdl1.2-dev
- libsdl2-dev
